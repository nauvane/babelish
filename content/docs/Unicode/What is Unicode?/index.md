Unicode is a computer standard that provides a unique code to each character in each language . To quote the Unicode site (Links to an external site.), 'Unicode provides a unique number for every character, no matter what the platform, program, or language is'.

All of these data are contained in character code charts (https://www.unicode.org/charts/) maintained by the Unicode Consortium. Here, for example, are the core data for languages that use the Latin script:
