Unicode supports 154 scripts, through which the writing systems of the world are mediated. With the publication of [Unicode 13.0.0](http://www.unicode.org/versions/Unicode13.0.0/) on 10 March 2020, Unicode now supports [154 scripts](https://www.unicode.org/standard/supported.html). 

This is how Unicode defines a script: "A collection of letters and other written signs used to represent textual information in one or more writing systems. For example, Russian is written with a subset of the Cyrillic script; Ukranian is written with a different subset. The Japanese writing system uses several scripts" ([**Glossary of Unicode terms**](https://www.unicode.org/glossary/#script)). 

### Latin script

This is the script that you are likely to use most often: it is the one in which English and many European languages are written.
